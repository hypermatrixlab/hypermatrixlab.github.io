HyperMatrix Lab Dashboard

Manage the lab website — update About Us, news, team members, and publications.

Setup (one-time) — choose one option:

Option A: Classic Token (easiest)

  1. Go to GitHub → Settings → Tokens (classic)
  2. Name it Lab Dashboard, set expiry as you wish
  3. Check the repo scope (full control of private repositories)
  4. Click Generate token and paste it below

Option B: Fine-grained Token (more secure)

  1. Go to GitHub → Settings → Fine-grained tokens
  2. Click "Generate new token"
  3. Name it Lab Dashboard, set expiry as you wish
  4. Under Repository access, select "Only select repositories" and choose hypermatrixlab.github.io
  5. Under Permissions → Repository permissions, find "Contents" in the list and change the dropdown from "No access" to "Read and write"
  6. Click Generate token and paste it below
